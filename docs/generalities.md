# CargoSnS 

Es una plataforma que permite a las personas o empresas que tengan mercancías pero no cuenten con espacio para almacenarlas, encontrar lugares seguros y confiables para almacenar sus mercancías. A su vez permite a las personas o empresas que tengan espacio disponible para almacenar mercancias, ofrecerlo a las personas o empresas que necesiten almacenar sus mercancias. Por otra parte le permite a transportistas independientes o empresas proveedoras de flota encontrar rutas y cargas para transportar.
